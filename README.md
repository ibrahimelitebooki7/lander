# Carlm — Landing Page

A modern, responsive landing page for **Carlm**, an intelligent AI assistant. Built with vanilla HTML, CSS, and JavaScript for maximum performance and zero build steps.

## 🚀 Quick Start

### Option 1: Open directly
1. Clone or download this repository.
2. Open `index.html` in your browser.
   - **Live Server recommended**: `npx live-server` for auto-reload.

### Option 2: Deploy to Vercel
1. Push this repo to GitHub.
2. Import it into [Vercel](https://vercel.com) — no configuration needed.
3. Vercel automatically detects `vercel.json` and serves the static site.
